How to read these results

BB-001 calibrates. BB-003 starts the real corpus.

Two scenarios are not a final leaderboard. BB-001 proves the harness (argparse bugfix), BB-003 adds a real feature task (JSON export). Together they establish the matrix shape and telemetry pipeline for future scenarios.
